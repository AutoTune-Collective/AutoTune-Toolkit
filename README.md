## About

**Auto-Tune Toolkit** gives you instant access to studio-grade vocal assets. Activate **Real-Time Pitch**, **Harmony Engine**, **Formant Shift**, load **custom scales**, and use **DAW configs** — all in one click.

## What's included

- `presets/` — Pop, Trap, Rock, Anime vocals
- `scales/` — Major, Minor, Pentatonic, Custom
- `effects/` — Reverb, Delay, Chorus, Saturation
- `configs/` — FL Studio, Ableton, Logic Pro profiles
- `guides/` — 30-second setup, mic tuning tips
- `assets/` — GUI themes, vocal previews

## Download & install (archive delivery)

1. Click **Launch Auto-Tune Now** — download starts (`.zip`).
2. Extract to `C:\AutoTune-Toolkit`.
3. Run `AutoTuneLauncher.exe` 
4. Open your DAW → presets auto-load.
> Pro tip: Use 48kHz + XLR mic for best results.

## Usage notes & best practices

- Test in DAW project first.
- Save original vocal chain.
- Use condenser mic + pop filter.
- Latency issues? Enable Low-Latency Mode.
